Latest Version

Pre-alpha 2.0 -- 4/2/2017 Version pre-alpha 2.0 contains many significant upgrades over the prior iterations of the websites. In version 2.0, a much more user friendly interface was layed out including breaking the site up into several interconnected webpages as opposed to everything being on one central webpage. The incorporation of several bootstrap elements (such as the navigation bar) allows for a much better and more aesthetically pleasing design. More work is also done in the CSS coding to make a generally nicer looking web design without tacky or clashing styles. Version pre-alpha 2.0 will be released to the public on GitHub Pages and will most likely be the operating version of the site for a while as we look into working on potential future updates and start other projects. Please bear with us until we are able to further polish the site and initialize more features. Thank you for your patience!
